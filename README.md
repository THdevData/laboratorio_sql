# 📚 Laboratório SQL

Projeto de estudos voltado para a construção de um banco de dados relacional utilizando **PostgreSQL**, com foco em modelagem, boas práticas, consultas SQL e recursos avançados da linguagem.

O objetivo é desenvolver um ambiente de aprendizado que evolua gradualmente, simulando um projeto real de banco de dados utilizado em empresas.

## 🎯 Objetivos

* Aprender modelagem de bancos de dados relacionais.
* Aplicar boas práticas de desenvolvimento em PostgreSQL.
* Praticar consultas SQL do nível básico ao avançado.
* Explorar recursos como Views, Functions, Procedures, Triggers e Índices.
* Criar um banco de dados rico em informações para resolução de desafios e estudos.

## 🛠️ Tecnologias

* PostgreSQL
* pgAdmin
* Visual Studio Code
* Git e GitHub
